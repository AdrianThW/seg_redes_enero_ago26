## Análisis del script

Se ejecuta el script sin argumentos:

```bash
python3 ende.py
```

Output:

```
Usage: ende.py (-e/-d) [file]
```

También se puede ver la ayuda:

```bash
python3 ende.py -h
```

Output:

```
Usage: ende.py (-e/-d) [file]
Examples:
  To decrypt a file named 'pole.txt', do: '$ python ende.py -d pole.txt'
```

---

## Interpretación

- `-e` → encriptar
    
- `-d` → desencriptar
    
- El script requiere:
    
    - un archivo
    - una contraseña

Se ejecuta:

```bash
python3 ende.py -d flag.txt.en
```

Se introduce la contraseña (contenido de `password.txt`):

```

```
