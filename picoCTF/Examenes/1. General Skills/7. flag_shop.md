
#### Description

There's a flag shop selling stuff, can you buy a flag? [Source](https://challenge-files.picoctf.net/c_fickle_tempest/66a0d80bfdedc5f74bdd52c50da2e5d7bf40c5634fd456b103ac74c006bf45e4/store.c). Connect with nc fickle-tempest.picoctf.net 63508.


Se accede al programa:

```bash
nc fickle-tempest.picoctf.net 63508
```


Se intenta comprar una gran cantidad del item barato:

```text
Cantidad: 2400000
```

Output:

```
The final cost is: -2134967296
```

### Explicación

- El cálculo del precio excede el límite de un entero de 32 bits
- El número se vuelve negativo
## ¿Qué es un Integer Overflow?

Cuando un número supera el límite permitido por su tipo de dato.

Ejemplo en C (int de 32 bits):

```
Máximo:  2,147,483,647
```

Si se excede:

el valor "da la vuelta" (overflow)

El sistema calcula:

```
precio_total = cantidad * precio
```

Pero debido al overflow:

```
precio_total = número negativo
```

Luego:

```
balance = balance - precio_total
```

 Restar un número negativo = sumar dinero

El balance se vuelve enorme:

```
Your current balance after transaction: 2134968396
```
