## Description

Connect to this PostgreSQL server and find the flag!

---

## Conexión a la base de datos

Se utiliza `psql` para conectarse al servidor:

```bash
psql -h saturn.picoctf.net -p 54941 -U postgres pico
```

### Explicación de parámetros

- `-h` → host (servidor remoto    
- `-p` → puerto
- `-U` → usuario (postgres)
- `pico` → nombre de la base de datos

Después se ingresa la contraseña proporcionada.
## Exploración de la base de datos

Una vez dentro, se pueden usar comandos especiales de `psql`.
### Listar tablas

```sql
\dt
```

Output:

```
List of relations
 Schema | Name  | Type  |  Owner   
--------+-------+-------+----------
 public | flags | table | postgres
```

### Explicación

- Existe una tabla llamada `flags`
- Está en el esquema `public`

## Consulta de datos

Se realiza una consulta SQL:

```sql
SELECT * FROM flags;
```

Output:

```
 id | firstname | lastname  | address
----+-----------+-----------+----------------------------------------
  1 | Luke      | Skywalker | picoCTF{L3arN_S0m3_5qL_t0d4Y_73b0678f}
  2 | Leia      | Organa    | Alderaan
  3 | Han       | Solo      | Corellia
```

---

## Identificación de la flag

La flag se encuentra en la columna `address` del primer registro.

## Comandos útiles en psql

- `\dt` → listar tablas
- `\d nombre_tabla` → ver estructura de tabla
- `\h` → ayuda SQL
- `\?` → ayuda de comandos de psql

Se logró acceder directamente a una base de datos PostgreSQL y consultar una tabla que contenía la flag.
