#### Description

Welcome to the challenge! In this challenge, you will explore a web application and find an endpoint that exposes a file containing a hidden flag. The application is a simple blog website where you can read articles about various topics, including an article about API Documentation. Your goal is to explore the application and find the endpoint that generates files holding the server’s memory, where a secret flag is hidden. The website is running [picoCTF News](http://verbal-sleep.picoctf.net:57727/).

La pagina tenia un apartado de documentacion de APIs, al descargar el archivo .heapsnapshot dentro del archivo venia la flag.

Un `.heapsnapshot` es un archivo que contiene una instantánea detallada de la memoria heap (montón) de JavaScript en un momento específico, utilizada para diagnosticar fugas de memoria y optimizar el rendimiento. Muestra todos los objetos en memoria, sus relaciones y tamaños, y se genera principalmente usando [Chrome DevTools](https://developer.chrome.com/docs/devtools/memory-problems/heap-snapshots?hl=es-419) o navegadores basados en Chromium.
