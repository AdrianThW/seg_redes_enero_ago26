### Referencias

Un **PHP webshell** es un **script malicioso escrito en PHP** que un atacante sube a un servidor web para poder **controlarlo remotamente desde el navegador**.

En pocas palabras:  
es como una **terminal escondida dentro de una página web**.

---

## ¿Cómo funciona?

1. El atacante encuentra una vulnerabilidad (por ejemplo, una subida de archivos sin validación).
    
2. Sube un archivo `.php` malicioso al servidor.
    
3. Accede a ese archivo desde el navegador.
    
4. Desde ahí puede ejecutar comandos del sistema como si estuviera en una terminal.
    

Un ejemplo muy simple (educativo) sería algo como:

<?php  
system($_GET['cmd']);  
?>

Si el atacante visita:

http://victima.com/shell.php?cmd=ls

El servidor ejecutará `ls` y mostrará el resultado.
