### Referencias

Las expresiones regulares (regex o regexp) son secuencias de caracteres que definen patrones de búsqueda, fundamentales en programación para validar, buscar, extraer o reemplazar texto. Permiten automatizar la manipulación de cadenas de caracteres, siendo vitales para validar formatos como correos electrónicos, números de teléfono o contraseñas.

Conceptos Clave de Expresiones Regulares:

- Caracteres Literales: Coinciden exactamente con el texto (ej. busca "abc").
    
- Metacaracteres (Especiales):
    
- : Coincide con cualquier carácter excepto nueva línea. y : Anclas que indican el inicio o final de una cadena, respectivamente. : El elemento anterior aparece 0 o más veces. : El elemento anterior aparece 1 o más veces. : El elemento anterior es opcional (0 o 1 vez). : Define un conjunto de caracteres (ej. para cualquier dígito). : Operador lógico OR (ej. busca "a" o "b").
    
- Cuantificadores: Definen cuántas veces debe aparecer un elemento (ej. exactamente n veces, al menos n veces).
    
- Escape: Se usa la barra invertida para buscar caracteres especiales como texto literal (ej. para buscar un punto real).
