### Referencias 

## ¿Qué es John the Ripper?

**John the Ripper** es una herramienta de auditoría de seguridad utilizada para probar la fortaleza de contraseñas. Es ampliamente usada en ciberseguridad para:

- Detectar contraseñas débiles.
    
- Realizar pruebas de penetración (pentesting).
    
- Auditar sistemas en entornos controlados.
## ¿Cómo funcionan los tokens (JWT)?

El tipo de token que estás manipulando es un **JSON Web Token (JWT)**.

Un JWT es un estándar abierto definido en el RFC 7519 para transmitir información de forma segura entre dos partes como un objeto JSON.
