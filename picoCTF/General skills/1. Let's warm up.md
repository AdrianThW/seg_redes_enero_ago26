
## Descripción

If I told you a word started with 0x70 in hexadecimal, what would it start with in ASCII?

## Solución

1. [From Hex - CyberChef](https://gchq.github.io/CyberChef/#recipe=From_Hex\('Auto'\)&input=MHg3MAo)

Usando Python

```
int(0x70)
112
chr(123)
'p'
ord('p')
112
```
