Can you crack the password to get the flag?Download the password checker [here](https://artifacts.picoctf.net/c/14/level2.py) and you'll need the encrypted [flag](https://artifacts.picoctf.net/c/14/level2.flag.txt.enc) in the same directory too.

```
AdrianThW-picoctf@webshell:~$ nano level2.py
AdrianThW-picoctf@webshell:~$ python level2.py 
Please enter correct password for flag: python3
That password is incorrect
AdrianThW-picoctf@webshell:~$ python level2.py 
Please enter correct password for flag: 4ec9
Welcome back... your flag, user:
picoCTF{tr45h_51ng1ng_9701e681}
AdrianThW-picoctf@webshell:~$ 
```

En el código la password  viene con el formato hexadecimal, hay que hacer el calculo de cada cadena para encontrar la contraseña

```
chr(0x34) + chr(0x65) + chr(0x63) + chr(0x39)
```
