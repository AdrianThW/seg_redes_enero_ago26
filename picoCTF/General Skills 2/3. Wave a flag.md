## Descripción

Can you invoke help flags for a tool or binary? This program has extraordinarily helpful information..

## Solución

```

AdrianThW-picoctf@webshell:~$ chmod +x warm
AdrianThW-picoctf@webshell:~$ ./warm
Hello user! Pass me a -h to learn what I can do!
AdrianThW-picoctf@webshell:~$ ./warm -h
Oh, help? I actually don't do much, but I do have this flag here: picoCTF{b1scu1ts_4nd_gr4vy_ac5832c}
```

Tambien es posible completar el reto con el comando strings

```
AdrianThW-picoctf@webshell:~$ strings warm | grep pico
Oh, help? I actually don't do much, but I do have this flag here: picoCTF{b1scu1ts_4nd_gr4vy_ac5832c}
AdrianThW-picoctf@webshell:~$ 
```
