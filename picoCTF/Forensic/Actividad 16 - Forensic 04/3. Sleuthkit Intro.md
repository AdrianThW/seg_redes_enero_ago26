## ¿Qué es una imagen de disco?

Una imagen de disco (`.img`) es una copia exacta de un disco duro o partición.  
Incluye:

- Tabla de particiones (MBR o GPT)
    
- Sistema de archivos
    
- Archivos internos
    

Este tipo de archivos se usa en análisis forense para investigar sistemas sin modificarlos.

---

## Análisis con mmls

Se ejecuta:

```bash
mmls disk.img
```

Output:

```
DOS Partition Table
Offset Sector: 0
Units are in 512-byte sectors

      Slot      Start        End          Length       Description
000:  Meta      0000000000   0000000000   0000000001   Primary Table (#0)
001:  -------   0000000000   0000002047   0000002048   Unallocated
002:  000:000   0000002048   0000204799   0000202752   Linux (0x83)
```

---

## Explicación del output

### DOS Partition Table

Indica que el disco usa el esquema **MBR (Master Boot Record)**.

---

### Units are in 512-byte sectors

Todos los valores están en **sectores de 512 bytes**, no en bytes directamente.

---

### Columnas

- **Slot** → número de partición
    
- **Start** → sector donde inicia
    
- **End** → sector donde termina
    
- **Length** → tamaño total en sectores
    
- **Description** → tipo de partición
    

---

### Particiones encontradas

#### 1. Meta

```
Primary Table (#0)
```

- Contiene la tabla de particiones
    
- No es una partición real
    

---

#### 2. Unallocated

```
0000000000 - 0000002047
```

- Espacio vacío del disco
    
- No contiene sistema de archivos
    

---

#### 3. Partición Linux

```
Start: 2048
End: 204799
Length: 202752
Type: Linux (0x83)
```

- `0x83` → tipo de partición Linux
    
- `Length: 202752` → **este es el dato que pide el reto**
