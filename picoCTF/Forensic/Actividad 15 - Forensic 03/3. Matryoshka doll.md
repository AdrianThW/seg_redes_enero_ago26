#### Description

Matryoshka dolls are a set of wooden dolls of decreasing size placed one inside another. What's the final one? Image: [dolls.jpg](https://challenge-files.picoctf.net/c_wily_courier/1709f471bed1c90c7b343b2d6a4f0d1257abed9ef467457d6438adb71be0b37c/dolls.jpg)

Al usar Strings no nos arroja ningun resultado.

Hay que instalar una herramienta que se llama binwalk.

Al usar la herramienta hay que desempaquetar la imagen sucesivamente hasta que encontremos la flag.

```
binwalk dolls.jpg

unzip dolls.jog

open 2_c.jpg
```

### Referencias 

Binwalk es una herramienta de línea de comandos diseñada para analizar, realizar ingeniería inversa y extraer archivos de imágenes de firmware de dispositivos IoT, enrutadores y otros sistemas embebidos.

Es ampliamente utilizada en seguridad para identificar componentes ocultos dentro de archivos binarios.
