#### Description

Decode this [message](https://challenge-files.picoctf.net/c_fickle_tempest/5593bde2464fd1977048cba02daa086d1fbc0fada1ef8521169ccdad2f19c6fc/message.wav) from the moon.

Al abrir el archivo solo se escucha estática, parece que el archivo contiene alguna codificacion utilizando algun metodo.

### Solucion:

QSSTV

Este programa permite **convertir señales SSTV contenidas en audio en imágenes**.

```
apt-get install qsstv
```

Crear dispositivo virtual de audio:

```
pactl load-module module-null-sink sink_name=virtual-cable
```

Primero iniciar la recepción en QSSTV presionando **Play**.

Luego reproducir el archivo de audio:

```
paplay -d virtual-cable message.wav
```

QSSTV recibe la señal y comienza a reconstruir la imagen transmitida.

Al finalizar la transmisión se obtiene la imagen que contiene la bandera del reto.
### Referencias

##### Slow-Scan Television (SSTV)

Slow-scan television es un método de transmisión de imágenes que envía **imágenes estáticas a través de señales de audio**.
