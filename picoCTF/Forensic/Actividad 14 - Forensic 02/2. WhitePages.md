Hay que crear un script para decifrar el codigo, en este caso sera en python

```
from pwn import *

with open("whitepages.txt", "rb") as bin_file:
    data = bytearray(bin_file.read()) 
    data = data.replace(b'\xe2\x80\x83', b'0')
    data = data.replace(b'\x20', b'1')
    data = data.decode("ascii")
    print unbits(data)
```

### Referencias

**Pwntools** es un framework y librería de Python diseñado específicamente para el desarrollo de exploits, la seguridad informática y la explotación de binarios (binary exploitation). Es extremadamente popular en las competiciones de **CTF** (Capture The Flag) debido a su capacidad para agilizar el desarrollo de prototipos y hacer que la creación de exploits sea más sencilla y legible.
