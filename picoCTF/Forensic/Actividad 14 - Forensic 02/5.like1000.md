#### Description

This [.tar file](https://challenge-files.picoctf.net/c_fickle_tempest/8014f84bda2778858cab85943405ae464bd01faed6c0ee344fa8b85c29260ff4/1000.tar) got tarred a lot.

El archivo fue empaquetado 1000 veces para "esconder"  la flag, la pista que nos marca el reto es hacer un script para no perder el tiempo desempaquetando todos los "tar".

El script es el siguiente:

```
adrian@ThW:~/Documents/adicional_seg$ for i in {1000..1}; do tar -xvf $i.tar; done
999.tar
filler.txt
998.tar
filler.txt
997.tar
```

Cuando termine estará la flag en el una imágen PNG.
